# yt-dlp-gui

A desktop GUI wrapper for [yt-dlp](https://github.com/yt-dlp/yt-dlp), built with Tauri and Angular.

This project aims to make common `yt-dlp` workflows easier to use from a desktop interface while keeping the power and reliability of the original command-line tool underneath. The goal is not to hide what `yt-dlp` can do, but to provide a friendly place for everyday tasks like inspecting media, choosing formats, managing downloads, and keeping track of download progress.

## Project Status

This is an early work-in-progress. The app is currently being shaped around a clean frontend structure, a Tauri command layer, and a test/lint/build workflow before the main download features are added.
